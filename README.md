## Usage
```bash
cd $HOME
git clone https://github.com/arkovyx/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh

## OR ##
## Use auto-yes (you won't be asked to select any packages, everything will be pre-selected)
./install.sh --yes
```
